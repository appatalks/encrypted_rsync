# encrypted_rsync
Rsync to LUKS

Simplistic script to Rsync from Local Machine to Target Host's LUKS Volume.
<p>
1) Send Key <br>
2) Mount and Decrypt LUKS Volume <br>
3) Delete Key <br>
4) Sync <br>
5) Umount / Encrypt LUKS Volume <p>

No error checking at this time. Should be easy to impliment. <p>
